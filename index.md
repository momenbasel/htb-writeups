## Quick Start

### For Beginners
1. Start with [Easy machines](machines/easy/) - they teach fundamentals
2. Follow the [Machine Approach Guide](resources/methodology/) for a systematic method
3. Use the [OSCP Prep](resources/cert-prep/) list if you are studying for certs
4. Explore the [Skill Trees](/htb-writeups/skill-trees/) to plan your learning path

### For Intermediate Players
1. Use the [Machine Finder](/htb-writeups/finder/) to filter by technique
2. Work through a [ProLab](prolabs/) (start with Dante)
3. Study the [Attack Paths](/htb-writeups/attack-paths/) for machines you have completed
4. Attempt [Sherlock](sherlocks/) challenges for blue team skills

### For Advanced Players
1. Target [Insane machines](machines/insane/) and Hard challenges
2. Complete [Cybernetics or APTLabs](prolabs/) ProLabs
3. Use the [Knowledge Graph](/htb-writeups/graph/) to find technique gaps
4. Write and contribute your own writeups
